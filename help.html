BA Banner Generator v1.5

Converts Figma banner frames into production-ready HTML5 animated banners, packaged as a ZIP ready for trafficking.

What it does

The plugin scans the current Figma page, finds banner frames organised by size (300x250, 728x90, etc.), exports every layer as a PNG, and builds an animated HTML5 banner that cross-fades between scenes. If your file uses Sections (one per funnel/copy variation), it exports a separate banner per section and packages everything into a structured master ZIP with preview pages.

Multiple scenes

To create an animated banner with multiple scenes, place several frames of the same size side by side on the page. The plugin detects them automatically and cross-fades between them.

💡
Scene frames don't need special names — the plugin groups all frames of the same size together. Just keep them on the same page.

Step-by-step workflow

1
📁 Pick your project folder
Click Project and select the root folder of your campaign. The plugin will auto-load optimised video files from optimized-videos/ inside that folder.
2
↺ Scan the Figma page
Click Scan. The plugin reads the current page, finds all banner frames and sections, reads the Control Panel, and populates the UI.
3
✓ Select sizes to export
All found sizes are checked by default. Uncheck any you don't need. Use Unselect All / Select All to toggle quickly.
4
🎬 Load video files (if needed)
If your banners have a video layer, the Video Files panel appears after scanning. If a project folder is set and videos were found automatically, the buttons will already show the filenames in green. Otherwise pick each MP4 manually.
5
⚙ Review settings
Expand Settings to set the click URL, scene duration (ms), fade time (ms), vertical code and campaign date. These are used in the exported filename and banner behaviour.
6
▶ Click "Convert HTML5 Banners"
The export overlay opens. A progress bar and elapsed timer show you how far along the process is. You can click ■ Stop at any time to cancel.
7
⬇ Review the log and download
Read the export log for any warnings (missing videos, hidden layers). When satisfied, click ⬇ Download ZIP. The master ZIP contains all banners organised by funnel and copy code, plus preview.html pages.

Video file setup

For the auto-load feature to work, organise your project folder like this:

📂
my-campaign/
  optimized-videos/
    casino/
      casino-tof-optimized.mp4
      casino-mof-optimized.mp4
      casino-bof-optimized.mp4

The subfolder name must match the vertical (e.g. casino, sports). Each file must contain TOF, MOF or BOF anywhere in its filename (case-insensitive).
